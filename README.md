# 달로썸 원고 검수기 v3.6 - 검수 후 생성 플로우

사용자가 원한 흐름을 반영한 검수 전용 버전입니다.

## 핵심 흐름

1. 원고 검수
2. 검수 후 달로썸 8가지 도입 중 하나 선택
3. 선택한 도입 방식으로 도입 문단 생성
4. 홈페이지 정보 유무 선택
5. 홈페이지 정보가 있으면 입력된 정보만 반영해 마무리 생성
6. 홈페이지 정보가 없으면 철학/장점을 임의 생성하지 않고 정보형 또는 일반 철학형으로 대체 마무리

## 주의

- 웹툰형은 실제 이미지가 아니라 컷 구성안만 생성합니다.
- 홈페이지 정보가 없으면 업체명, 원장명, 강점은 임의로 만들지 않습니다.
