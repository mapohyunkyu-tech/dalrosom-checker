# 달로썸 원고 검수기 v3.7 - 리라이트 선택 생성 버전

v3.6에서는 검수 후 도입/마무리 리라이트가 항상 표시됐지만, v3.7에서는 필요한 경우에만 생성하도록 바꾼 버전입니다.

## v3.7 수정점

- 도입 리라이트 생성 체크박스 추가
- 마무리 문단 생성 체크박스 추가
- 둘 다 꺼져 있으면 검수 결과만 표시
- 원고가 통과권이면 "현재 도입과 마무리 유지 가능" 안내
- 웹툰형 안내는 웹툰형을 선택하고 도입 생성이 켜진 경우에만 표시

## 사용 흐름

1. 원고 붙여넣기
2. 검수 시작
3. 점수와 문제점 확인
4. 필요할 때만 도입 리라이트 또는 마무리 문단 생성 체크
