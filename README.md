# 달로썸 원고 검수기 v4.0

## 핵심 변경

v4.0은 v3.9 감정유지 버전에 **분량 조건 제어**를 추가한 버전입니다.

### 추가 기능

1. GPTs용 초안 프롬프트 생성 시 분량 조건 자동 삽입
   - 1000자
   - 1500자
   - 2000자
   - 3000자
   - 직접 입력

2. 공백 제외 / 공백 포함 기준 선택

3. 문단 설정 선택
   - 분량 우선, 문단 수 자연 조절
   - 3문단
   - 4문단
   - 5문단
   - 6문단 이상

4. 테스트 분량이 있는 경우 기존 5문단/문장 수 설정보다 분량 조건 우선

5. 원고 검수 모드에서 테스트 분량 기준에 맞춰 권장 최소/최대 글자수 자동 반영

## 사용 흐름

1. GPT 조사 프롬프트 생성
2. GPT로 자료 조사
3. 사용자가 링크 직접 확인
4. 원고 설계 모드에 확인 자료 붙여넣기
5. 분량 조건 선택
6. GPTs용 초안 프롬프트 생성
7. GPTs에서 초안 작성
8. 원고 검수 모드에서 1차 검수
9. Claude용 지시문으로 말맛만 다듬기
10. 최종 검수 후 90점 이상이면 잠금

## 실행 방법

```bash
pip install -r requirements.txt
streamlit run app.py
```
