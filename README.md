# 달로썸 원고 검수기 v10.0.19

실사용 OK판 패치.

## 핵심 수정
- ⑧ 검수 점수/등급 충돌 완화
  - 세부 업종 필수 기준이 남아 있는데 100점으로 뜨는 문제 보정
  - 필수 기준 1개 보강 여지가 있으면 100점 판정을 제한
- 사람화 수정 전/후 오류 보정
  - “하는 데 도움이 됩니다” → “하는 데 판단하는 데 참고가 됩니다”처럼 더 어색해지는 자동 치환 차단
  - 선택사항 수준의 문장 제안만 있을 때는 100점 원고를 과하게 낮추지 않음
- 붙여넣기 라벨 처리 보정
  - `본문:` 라벨이 실제 첫문장으로 잡히지 않도록 처리
  - `최종 제목:` 라벨 제목 인식 보강
- ⑩ 국어선생님 결과 공유용 TXT 다운로드 추가
  - CSV 외에 ChatGPT/Claude에 바로 보여줄 수 있는 txt 리포트 다운로드 가능
