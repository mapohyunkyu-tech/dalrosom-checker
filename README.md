# 달로썸 원고 검수기 v5.4

v5.4 변경점

- 첫문장 B등급 고민 포함 검수 추가
- 원고 검수 모드에 `B등급 고민 요약 / 첫문장 근거` 입력칸 추가
- 첫문장이 물음표로 시작해도 B등급 핵심 고민어가 약하면 감점
- B등급 고민 핵심어 예시: 효과 없음, 약 반응 없음, 부작용, 비용 불안, 기저질환, 비교 혼란, 선택 불안
- GPTs 프롬프트에 `첫 문장에는 B등급 고민 핵심어 1~2개 직접 반영` 조건 추가
- 첫문장 의문문 강제 옵션과 별도로, 실제 고민 포함 여부를 확인

실행

```bash
pip install -r requirements.txt
streamlit run app.py
```
