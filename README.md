# 달로썸 원고 검수기 v9.4.4

## v9.4.4 DB 후보 단순화 패치

이번 버전은 `⑧ 대량 샘플 테스트랩`에서 DB 후보 처리가 헷갈리는 문제를 줄인 핫픽스입니다.

### 핵심 변경

- `간단 모드` 추가
  - 저장 추천
  - 자동 보류
  - 검토만
- `앱 추천대로 안전 후보만 저장` 버튼 추가
  - 사용자가 후보를 하나씩 고르지 않아도 됨
  - 완전 금지/반복 감점 중 안전한 후보만 저장
  - 보류 후보는 저장하지 않음
- 짧은 단어 단독 후보 자동 보류
  - `무조건`
  - `직접`
  - `100%`
  - `최고의`
- 고급 모드는 접힘 처리
  - 직접 고르고 싶을 때만 열면 됨

### 사용법

1. `⑧ 대량 샘플 테스트랩`에서 샘플 생성·검수 실행
2. `간단 모드`의 저장 추천 개수 확인
3. 이상해 보이지 않으면 `앱 추천대로 안전 후보만 저장` 클릭
4. 자동 보류/검토 후보는 기본적으로 건드리지 않음

### 검사

`python -m py_compile app.py` 통과.
