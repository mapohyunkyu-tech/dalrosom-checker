# 달로썸 원고 검수기 v6.7

## 수정 내용

- 글 성격을 바꾸면 `말투 세부 지시` 기본 문구가 같이 바뀌도록 수정했습니다.
  - 일반 정보성
  - 대표 작성형 정보성
  - 매출 전환형 정보성
  - 홈판 후킹형
- 원고 목적 기본값을 빈칸으로 변경했습니다.
  - ① 의뢰 조건 입력의 원고 목적 기본값 제거
  - 원고 목적을 비워둬도 병원 블로그 목적이 자동 삽입되지 않게 수정
- 수동 검수 모드의 원고 목적도 빈값 선택이 가능하도록 변경했습니다.

## 실행

```bash
streamlit run app.py
```
