# 달로썸 원고 검수기 v10.0.5

v10.0.4 Claude 자연화 복붙 패키지 이후, 사용자가 실제 작업 순서대로 보기 쉽도록 탭 번호와 탭 순서를 재정렬한 버전입니다.

## 이번 수정

- 기능 추가 없이 사용 흐름 중심으로 번호 재정렬
- 기존 기능은 유지하되 탭 표시 순서를 실제 작업 순서로 변경
- 상단에 사용 흐름 안내문 추가
- ①/②/③처럼 화면 안내에 남아 있던 번호도 새 순서에 맞게 갱신

## 새 탭 순서

1. ① 클라이언트 프리셋·기본 조건
2. ② 의뢰 조건 입력·GPT 조사 프롬프트
3. ③ 조사 결과 붙여넣기·원고 설계
4. ④ 상품별 원고 엔진
5. ⑤ 사진자료 기자단 작성기
6. ⑥ 쇼츠 대본 엔진
7. ⑦ 홈피드 제목·썸네일 실험실
8. ⑧ 최종 원고 검수·Claude 패키지
9. ⑨ 사람화 수정 제안
10. ⑩ 국어선생님 퇴고
11. ⑪ 통합 출고 판정
12. ⑫ 전업 납품 대시보드
13. ⑬ 대량 샘플 테스트랩
14. ⑭ 이상 문장 DB 편집기
15. ⑮ 오류발견기
16. ⑯ 월관리 클라이언트
17. ⑰ 수익 작업대
18. ⑱ 견적서·단가표
19. ⑲ 작업 통계
20. ⑳ 샘플 포트폴리오
21. ㉑ 백업·복원

## 권장 사용 흐름

① 프리셋 불러오기/저장  
→ ② 의뢰 조건 입력  
→ ③ 조사 결과 붙여넣기  
→ GPTs 초안/Claude 자연화  
→ ⑧ 최종 원고 검수  
→ ⑩ 국어선생님  
→ ⑪ 통합 출고판정  
→ 필요 시 ⑫~㉑ 운영관리
