# 달로썸 원고 검수기 v4.8

v4.6 기반 오탐/표시 보정 버전입니다.

## 변경 사항

1. 위험표현 오탐 완화
- “무조건 더 좋다고 말하기 어렵습니다”처럼 단정 금지 문맥은 위험표현으로 보지 않도록 보정했습니다.
- “무조건 효과/개선/좋아짐/승소” 등은 계속 위험표현으로 감지합니다.

2. 조사 오류 수정
- 도입 리라이트 생성에서 “써마지을”처럼 받침 없는 키워드에 조사 오류가 나는 문제를 줄였습니다.

3. FAQ 도입 감지 개선
- 따옴표 질문이 2개 이상 나오거나 “질문을 많이 받습니다”, “물어보시는 분” 같은 표현이 있으면 ‘많이 묻는 질문 인용’으로 더 정확히 감지합니다.

4. 철학 반영 표시 개선
- 홈페이지 정보 반영 / 사용자 입력 철학 반영 / 정보형 마무리를 구분해서 표시합니다.

5. 키워드 반복 감점 완화
- 1500자 내외에서 키워드 10~12회는 큰 감점보다 자연스러움 경고 위주로 조정했습니다.

## 사용 순서

GPT 조사 → 링크 확인 → 원고 설계 모드 → GPTs 초안 → 원고 검수 → Claude 윤문 → 최종 검수


## v4.8 추가
- GPT 조사 프롬프트와 GPTs용 초안 프롬프트에 키워드 반복 제한을 추가했습니다.
- 1500자 내외 원고 기준 제목 포함 4~6회, 본문 3~5회 안팎을 목표로 하도록 조정했습니다.
- 원고 검수에서도 분량별 키워드 권장 횟수를 동적으로 계산해 과다 경고를 더 현실적으로 표시합니다.
