# 달로썸 원고 검수기 v10.0.14

v10.0.14는 원고 작성용 **글결 조종판** 업데이트입니다.
사진, 예약 카드, 하단 고지 모듈이 아니라 원고 본문 흐름만 조절합니다.

## 주요 변경

- ② 의뢰 조건 입력에 `대표 글결` 추가
- `보조 글결 1`, `보조 글결 2` 추가
- 글결 선택값을 원고 목적 자동 생성에 반영
- ③ GPTs 조사/초안 프롬프트에 `[원고 글결 적용 지시]` 반영
- ⑧ Claude 보강·자연화 패키지에 글결 유지 지시 반영
- ⑧ 앱 검수에서 대표 글결 반영 약함 감지

## 대표 글결 예시

- 증상 공감형: 달로썸 무릎 원고 같은 흐름
- 오해 반박형: 써마지처럼 흔한 오해를 반박하는 흐름
- 병원/업체 선택 기준형: 백내장처럼 병원 고르는 기준을 알려주는 흐름
- 장비보다 해석/판단 강조형: 장비 자체보다 분석·해석·상담 기준을 강조하는 흐름
- 대표자 주장형: 원장/변호사/대표의 관점과 철학이 드러나는 칼럼형 흐름

## 사용 방식

1. ②에서 분야, 글 성격, 주제, 키워드, 전환 목표 입력
2. 대표 글결은 `자동 추천` 또는 직접 선택
3. 필요하면 보조 글결 최대 2개 선택
4. ③에서 GPTs 초안 프롬프트 생성
5. ⑧에서 Claude 보강 패키지 생성 및 앱 검수
