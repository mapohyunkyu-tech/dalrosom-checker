# 달로썸 원고 검수기 v3 - 검수 전용 안정 버전

자동 자료수집 기능을 제거하고, 테스트 원고 검수에 필요한 항목만 안정적으로 점검하는 버전입니다.

## 왜 v3인가?

이전 버전의 자동 자료수집은 API 검색 결과 품질에 따라 편차가 컸습니다.
v3는 자료수집을 제외하고 아래 항목만 정확하게 봅니다.

- 제목 자동 인식
- 제목 키워드 여부
- 본문 키워드 횟수
- 공백 제외 글자수
- 소제목 수
- 도입부 공감/체크리스트 여부
- AI티 표현
- 위험표현
- 작성자 관점
- 마무리
