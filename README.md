# 달로썸 원고 검수기 v4.9

## 화법 선택 + 전 분야 감정 흐름 생성 버전

v4.9는 v4.8의 키워드 반복 제한 기능에 더해, 병원·법률·홈케어·학원·제품 등 전 분야에서 쓸 수 있는 **화법 선택**과 **감정 흐름 설계**를 추가한 버전입니다.

## 추가/개선 사항

1. 화법 선택 방식 개선
   - 자동 추천
   - 직접 선택
   - 최종 적용 화법 표시

2. 화법 종류 확장
   - 일상 불편형
   - 판단 혼란형
   - 감정 직면형
   - 억울함 공감형
   - 비교 고민형
   - 비용 불안형
   - 선택 불안형
   - 후회 예방형
   - 전문가 안내형
   - 질문 응답형

3. 전 분야 감정 흐름 설계 추가
   - 감정 도입 첫문장 후보 3개
   - 본문 고민 브릿지 후보 3개
   - 마무리 재연결 문장 후보 2개

4. GPTs용 초안 프롬프트 강화
   - 도입부 첫 문장을 정보 설명으로 시작하지 않도록 지시
   - 선택 화법과 도입 8가지 방식을 함께 강제
   - B등급 고민을 도입뿐 아니라 본문 전환부에도 배치

5. Claude 윤문 지시 강화
   - 도입부 첫 2~3문장을 감정 도입 보호 문장으로 보고 정보형으로 바꾸지 않도록 지시

## 사용 흐름

1. GPT 조사 프롬프트 생성
2. GPT로 자료조사
3. 링크 확인 후 조사 결과를 원고 설계 모드에 붙여넣기
4. 자동 추천 화법 확인 또는 직접 선택
5. 감정 도입/본문 브릿지/마무리 재연결 문장 확인
6. GPTs용 초안 프롬프트 복사
7. 초안 작성 후 검수 모드에서 점수 확인
8. Claude에 윤문 지시문과 함께 보내기
9. 최종 검수 후 제출

## 실행

```bash
pip install -r requirements.txt
streamlit run app.py
```
