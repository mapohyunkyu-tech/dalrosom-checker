# 달로썸 원고 검수기 v5.0 화법 선택 표시 강화 버전

v4.9 기능은 유지하면서, 사용자가 못 찾기 쉬웠던 **화법 선택**을 조사/설계/검수 화면에 모두 보이도록 수정했습니다.

## 수정 내용

- ① GPT 조사 프롬프트 탭에 `희망 도입 화법` 선택 추가
  - 자동 추천
  - 일상 불편형
  - 판단 혼란형
  - 감정 직면형
  - 억울함 공감형
  - 비교 고민형
  - 비용 불안형
  - 선택 불안형
  - 후회 예방형
  - 전문가 안내형
  - 질문 응답형

- ② 원고 설계 모드에 `화법 선택` 섹션을 더 눈에 띄게 표시
  - 자동 추천 화법 / 최종 적용 화법 확인 가능

- ③ 원고 검수 모드 사이드바에 `현재 원고 화법` 선택 추가
  - 검수 참고용
  - 도입 리라이트와 Claude 지시 방향 확인에 사용

## 실행 방법

```bash
pip install -r requirements.txt
streamlit run app.py
```
