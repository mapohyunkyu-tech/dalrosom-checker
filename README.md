## v10.0.28 DB 라우터 선별 엔진
- 전체 감정·귀찮음·행동 DB와 사용자 이상문장 DB를 API에 한 번에 던지지 않고, 분야/사용처/글성격/글결/작성자 관점에 맞는 조각만 선별해 프롬프트에 삽입합니다.
- 병원·법률·금융·투자 등 민감 업종은 직접 체험처럼 보이는 몸 반응/후폭풍 표현을 제외하고, 비교 피로·확인 강박·돈 고민·상담 전 망설임 위주로 낮춰 사용합니다.
- 사용자 이상문장 DB는 현재 분야와 가까운 금지 패턴만 8~10개 수준으로 선별해 생성 단계에서 피하도록 합니다.
- 기존 복붙 프롬프트와 API 생성 모두 같은 DB 라우터 블록을 사용합니다.
