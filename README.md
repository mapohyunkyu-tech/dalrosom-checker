# 달로썸 원고 검수기 v4.6

v4.5 기반 수정 버전입니다.

## 변경 사항

1. 제목 출력 형식 강화
- 제목 후보 3개
- 최종 제목
- 본문
형식으로 분리 출력하도록 GPTs용 프롬프트에 강제 문구를 추가했습니다.

2. 홈페이지 정보 없을 때 마무리 금지 강화
- 본원은
- 저희 병원은
- 저희는
- 우리 병원은
- 대표원장은
- 환자 중심
- 정직한 진료
- 풍부한 경험
같은 병원 철학/장점/운영방침을 임의로 만들지 않도록 강화했습니다.

3. FAQ 도입 표기 오류 방지
- “2~3개”가 “23개”로 붙어 보이지 않도록 조사 프롬프트에 명시했습니다.

4. Claude 지시문 강화
- 홈페이지 정보가 없을 때 기관 철학/장점을 지어내지 말라는 조건을 더 강하게 넣었습니다.

## 사용 순서

GPT 조사 → 링크 확인 → 원고 설계 모드 → GPTs 초안 → 원고 검수 → Claude 윤문 → 최종 검수
