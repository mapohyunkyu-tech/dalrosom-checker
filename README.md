# 달로썸 원고 검수기 v5.6

v5.6 변경 사항

- ① GPT 조사 프롬프트 단계에 홈페이지/업체 정보 입력칸 추가
- 홈페이지 정보 있음 선택 시 조사 프롬프트에 원장/대표 소개, 철학, 장점 정리 요청 반영
- 홈페이지 정보가 없으면 본원/저희/철학/장점 문구 임의 생성 금지 유지
- 조사 결과 출력 형식에 홈페이지 정보 요약, 마무리 반영 문장 후보 추가
- ② 원고 설계 모드와 ③ 검수 모드의 홈페이지 정보 반영 흐름은 기존 유지

사용 순서

1. ① GPT 조사 프롬프트에서 주제, 키워드, 화법, 도입 방식, 첫문장 형태를 선택합니다.
2. 홈페이지 정보가 있으면 ① 단계에서 바로 붙여넣습니다.
3. 생성된 조사 프롬프트를 GPT에 넣고 자료조사를 받습니다.
4. 확인한 조사 결과를 ② 원고 설계 모드에 붙여넣어 GPTs용 초안 프롬프트를 만듭니다.
5. 최종 원고는 ③ 원고 검수 모드에서 확인합니다.
