# 달로썸 테스트 원고 검수기 v1

GPTs에서 받은 병원/법률/에스테틱 블로그 초안을 붙여넣고, 제목·도입·본문 SEO·AI티·위험표현·마무리·점수/예상 단가를 규칙 기반으로 검수하는 Streamlit 앱입니다.

## 실행 방법

```bash
pip install -r requirements.txt
streamlit run app.py
```

## v1 기능

- 원고 목적 선택
  - 포트폴리오용 샘플 원고
  - 마케팅 회사 테스트 원고
  - 실제 병원 광고 원고
  - 실제 법률 광고 원고
  - 일반 정보성 원고
- 분야 선택
  - 에스테틱 / 피부관리
  - 병원 / 의료
  - 법률
  - 기타 전문업종
- 제목 검수
- 도입부 검수
- 본문 SEO 검수
- AI티 표현 검수
- 의료/법률 위험표현 검수
- 어려운 용어 코너 추천
- 마무리 문단 검수
- 총점 및 예상 단가 출력

## 주의

이 앱은 법률·의료 광고 규정에 대한 최종 법적 판단 도구가 아닙니다. 실제 광고 게시 전에는 광고주, 의료기관, 법무법인, 전문가의 최종 검수를 거쳐야 합니다.
