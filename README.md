# 달로썸 원고 검수기 v10.0.24

## 패치 요약
- 도입 다양화/반복 방지 지시 추가: 같은 주제 반복 생성 시 `~을 알아보다 보면`, `~을 고민하는 분들이`, `가장 많이 묻는 질문은` 같은 시작 패턴을 피하고 상담 장면·오해 반박·대표자 판단 기준 등으로 변주.
- 본문 전개 패턴 분산 추가: 고민→원리→기준 흐름만 반복하지 않고 오해반박형, 상담장면형, 비용비교형, 실패원인형 등으로 흐름 선택.
- 업체명 없음 하드 차단 강화: 병원명/업체명 없음 상태에서 `저희 병원`, `본원`, `저희 클리닉`, `저희는` 등 기관 표현을 검수에서 감지하고 총점 상한 적용.
- 대표자 직접 서술형 검수 강화: `저는/제가` 중심 판단 문장이 부족하거나 `의료진은/상담에서는/일반적으로는` 같은 3자 표현이 과하면 작성자 관점 감점.
- Claude/국어선생님/사람화 수정본 조건 이탈 방지 문구 추가: 자연화 후에도 글결, 작성자 관점, 업체명 없음 조건, 전환 목표가 흐려지지 않도록 패키지에 보존 지시 삽입.
- v10.0.23 의료 감정선 오탐 보정 유지: `충분한지`, `충분한`, `불충분한`, `흥분한`을 `분한` 감정어로 잘못 잡지 않음.
