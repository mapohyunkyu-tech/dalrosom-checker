# 달로썸 원고 검수기 v10.0.6

## 이번 수정

v10.0.5 사용 순서 정리판을 기준으로, 마케팅 업체 실사용에 맞춰 **글 성격 8종**과 **글 성격별 마무리 판정 보정**을 추가했습니다.

## 글 성격 8종

1. 정보성
2. 업체/서비스 소개형
3. 선택 기준형
4. 문제 해결형
5. 매출 전환형
6. 후기/기자단형
7. 카페 자연글형
8. 홈피드 이슈형

기존 값은 자동 매핑됩니다.

- 일반 정보성 → 정보성
- 대표 작성형 정보성 → 업체/서비스 소개형
- 매출 전환형 정보성 → 매출 전환형
- 대표 작성형 + 매출 전환형 정보성 → 매출 전환형
- 홈피드 후킹형 / 홈피드 수익형 블로그 글 / 홈판 후킹형 → 홈피드 이슈형

## 마무리 판정 보정

- 글 성격이 `매출 전환형`이면 병원/업체 강점 + 상담 유도 마무리를 정상 범위로 봅니다.
- 글 성격이 `정보성`, `카페 자연글형`, `홈피드 이슈형`, `후기/기자단형`이면 마지막 문단의 업체/병원 소개 과밀을 주의로 봅니다.
- 글 성격이 `선택 기준형`, `문제 해결형`이면 업체 정보 반영은 가능하지만 마지막 문단 장점 과밀은 주의로 봅니다.
- 의료/법률/금융 보장 표현은 글 성격과 무관하게 계속 차단합니다.
