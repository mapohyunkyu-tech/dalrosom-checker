# 달로썸 원고 검수기 v7.6

## 핵심 변경

### 1. 병원/의료 작성자 관점 자동 변경
병원 / 의료 분야에서 주제와 핵심 키워드에 따라 작성자 관점을 자동 추천합니다.

예시:
- 써마지, 울쎄라, 인모드, 보톡스, 필러, 레이저 → 피부과 원장/전문의
- C-ARM, 주사치료, 영상유도, 통증, 척추, 관절, 디스크, 협착증 → 통증·척추관절 진료 병원 대표원장/전문의
- 임플란트, 교정, 사랑니, 치아 → 치과 대표원장/전문의
- 라식, 백내장, 녹내장 → 안과 원장/전문의
- 산부인과, 비뇨기과, 소아청소년과 키워드도 자동 추천

### 2. 대표 작성형 의료 말투 강화
글 성격이 `대표 작성형 정보성`이고 작성자 관점이 병원 원장/전문의 계열이면 말투 세부 지시에 아래 방향이 자동 반영됩니다.

- 대표원장/전문의가 진료실에서 환자에게 직접 설명하듯 작성
- 본문 2~3곳에 실제 설명 문장 삽입
  - “진료실에서 자주 듣는 질문입니다”
  - “환자분들이 가장 헷갈려하는 부분입니다”
  - “상담 때는 이 기준을 먼저 확인합니다”
- 치료 효과 보장, 과한 홍보, 가짜 경험담 금지
- 병원 장점/철학은 공식 홈페이지나 사용자가 제공한 자료가 있을 때만 반영

### 3. C-ARM 주사치료 같은 키워드 대응
`C-ARM 주사치료`를 병원/의료로 넣으면 피부과 원장 관점이 아니라 통증·척추관절 진료 의료진 관점으로 잡히도록 수정했습니다.

### 4. 홈피드 카테고리 엔진 유지
v7.5의 홈피드 카테고리별 이슈 수집 → 정리 → 홈피드화 구조는 그대로 유지했습니다.
