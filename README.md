# 달로썸 원고 검수기 v6.4

## 변경 사항

- Claude 복붙 지시문이 예전 화법/사용처를 들고 오는 문제를 줄이기 위해 동적 갱신 키를 적용했습니다.
- ② 원고 설계 모드의 GPTs 프롬프트/Claude 지시문 텍스트 영역이 현재 주제, 키워드, 사용처, 화법, 도입 방식, 첫문장 형태, 홈페이지 정보 기준으로 새로 갱신됩니다.
- ③ 원고 검수 모드에서 검수 후 바로 쓸 수 있는 `Claude용 윤문 지시문 · 현재 검수값 자동 갱신` 영역을 추가했습니다.
- 검수 모드 자동 적용값이 ① 기본값뿐 아니라 ② 최종 적용값(사용처, 분야, 키워드, 화법, 도입 방식, 제목 유형 등)을 우선 가져오도록 정리했습니다.
- Claude 지시문 상단에 이번 검수/설계 화면의 최신 조건을 명시해, Claude가 이전 템플릿의 화법명이나 사용처를 따라가지 않도록 했습니다.

## 사용법

1. ①에서 의뢰 조건을 입력하고 GPT 조사 프롬프트를 만듭니다.
2. GPT 조사 결과를 ②에 붙여넣고 원고 설계를 만듭니다.
3. GPTs 초안 작성 후 ③ 원고 검수 모드에 원고를 붙여넣고 검수합니다.
4. 검수 결과 아래의 `Claude용 윤문 지시문 · 현재 검수값 자동 갱신` 칸을 Claude에 복붙합니다.

## 설치

```bash
pip install -r requirements.txt
streamlit run app.py
```
