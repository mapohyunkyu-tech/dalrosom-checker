# 달로썸 원고 검수기 v10.0.7

## 사용 순서 UI 정리 핫픽스

v10.0.7은 기능 추가가 아니라, 실제 작업 흐름에서 헷갈리던 문구를 정리한 버전입니다.

### 수정 내용
- ③ 조사 결과 붙여넣기·원고 설계 탭에서 Claude용 윤문 지시문 기본형을 제거했습니다.
- ③은 GPTs 초안 프롬프트 생성 단계로만 정리했습니다.
- Claude 자연화 패키지는 ⑧ 최종 원고 검수·Claude 패키지 탭에서만 생성하도록 안내를 정리했습니다.
- ⑧의 원고 입력칸 이름을 `GPTs 초안 또는 Claude 수정본 붙여넣기`로 변경했습니다.
- `검수 시작` 버튼을 `앱 검수 실행`으로 변경했습니다.
- ⑧ 상단에 실제 사용 순서를 추가했습니다.
- 검수 후 아래에 나오는 Claude 지시문은 `검수 결과 반영 재수정용 Claude 지시문 · 선택`으로 바꿔, 필수 단계가 아니라 수정 필요 시 사용하는 칸임을 명확히 했습니다.

### 권장 사용 순서
1. ② 의뢰 조건 입력
2. ③ 조사 결과 붙여넣기·GPTs 초안 프롬프트 생성
3. GPTs에서 초안 생성
4. ⑧에 GPTs 초안 붙여넣기
5. ⑧ Claude 자연화 패키지를 Claude에 복붙
6. Claude 수정본을 다시 ⑧ 원고 칸에 붙여넣기
7. `앱 검수 실행`
8. ⑩ 국어선생님 퇴고
9. ⑪ 통합 출고 판정
