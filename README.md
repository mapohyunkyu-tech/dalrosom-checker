# 달로썸 원고 검수기 v9.4.3

## 업데이트 요약

v9.4.3은 v9.4.2에서 추가된 DB 후보 자동분류/승인 기능이 화면에서 헷갈리지 않도록 위치와 안내를 보강한 핫픽스입니다.

### 변경 사항

- 앱 제목을 v9.4.3으로 변경
- `⑧ 대량 샘플 테스트랩` 상단에 `DB 후보 승인판 · 애매한 건 여기서 봅니다` 안내 추가
- 샘플 실행 전에도 후보 창이 왜 비어 있는지 설명 표시
- `볼 후보` 필터에 `애매/보류만` 추가
- 보류 후보가 없을 때 필터를 넓히라는 안내 추가
- `자동 추천 DB 후보` 영역 설명 강화
- `python -m py_compile app.py` 통과

## 보는 법

1. `⑧ 대량 샘플 테스트랩`으로 이동합니다.
2. 테스트할 상품/분야/샘플 수를 고릅니다.
3. `대량 샘플 생성·검수 실행` 버튼을 누릅니다.
4. `자동 추천 DB 후보`에서 `승인 추천만`을 먼저 봅니다.
5. 애매한 문장만 보고 싶으면 `볼 후보`를 `애매/보류만`으로 바꿉니다.
6. 확실한 것만 저장하고, 애매한 것은 보류하거나 저장하지 않습니다.
