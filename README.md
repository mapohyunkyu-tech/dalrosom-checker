# 달로썸 원고 검수기 v6.1

## 변경 사항

- 자동 추천 도입 방식 로직 조정
- `확인할 점`, `체크할 것`, `주의사항`, `선택 전`, `계약 전`, `시술 전`, `수술 전` 같은 주제는 체크리스트 도입을 우선 추천
- 조사자료에 `비교`, `가격 비교`, `A/B` 같은 단어가 섞여 있어도 주제 자체가 비교글이 아니면 비교표로 과하게 빠지지 않도록 수정
- `차이`, `비교`, `VS`, `장단점`, `둘 중 무엇`처럼 명확한 비교 주제일 때만 비교표를 우선 추천
- v6.0의 검수조건 자동 불러오기 기능 유지

## 실행

```bash
pip install -r requirements.txt
streamlit run app.py
```
