# 달로썸 원고 검수기 v6.0

## 변경사항
- 자동 추천 선택칸은 화면에 `자동 추천`으로 유지됩니다.
- 제목 유형/도입 8가지 방식은 자동 추천을 선택하면 내부에서 추천값을 최종 적용하고, 화면에는 추천값과 최종 적용값을 함께 보여줍니다.
- ③ 원고 검수 모드에 `①/② 입력값으로 검수 조건 자동 적용` 옵션을 추가했습니다.
- 검수 모드에서 주제, 분야, 키워드, 화법, 도입 방식, 첫문장 형태, 분량, 키워드 납품 기준을 다시 입력하지 않아도 됩니다.
- 수동 검수가 필요하면 체크를 끄고 기존처럼 직접 입력할 수 있습니다.
