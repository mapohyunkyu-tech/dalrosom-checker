# 달로썸 원고 검수기 v4.2

## 변경 내용

- `① GPT 조사 프롬프트` 탭에 **희망 도입 8가지 방식** 선택칸을 추가했습니다.
  - 자동 추천
  - 체크리스트
  - 비교표
  - 대화체
  - 뉴스 기사
  - 질문형
  - 많이 묻는 질문 인용
  - 알짜정보 예고
  - 웹툰/장면 구성

- `② 원고 설계 모드`에 **달로썸 도입 8가지 방식 선택** 기능을 추가했습니다.
  - 프로그램이 주제/자료/도입화법을 보고 자동 추천합니다.
  - 사용자가 직접 다른 도입 방식으로 변경할 수 있습니다.

- GPTs용 초안 프롬프트에 아래 내용이 자동 반영됩니다.
  - 선택한 도입화법
  - 선택한 달로썸 도입 방식
  - 도입 방식별 세부 지시

- Claude용 윤문 지시문에 아래 내용이 자동 반영됩니다.
  - 도입화법 변경 금지
  - 달로썸 도입 방식 변경 금지
  - 고민/감정 문장 삭제 금지

## 실행 방법

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 사용 흐름

1. GPT 조사 프롬프트 생성
2. 링크 확인 후 자료 붙여넣기
3. 자료등급/고민패턴/도입화법/도입 8가지 선택
4. GPTs용 초안 프롬프트 생성
5. GPTs 초안 작성
6. 원고 검수
7. Claude 말맛 다듬기
8. 최종 검수
