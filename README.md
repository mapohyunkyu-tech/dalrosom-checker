# 달로썸 원고 검수기 v5.5

Streamlit 기반 블로그 원고 설계/검수 도구입니다.

## 실행
```bash
pip install -r requirements.txt
streamlit run app.py
```

## v5.5 변경점
- ② 원고 설계 모드에 홈페이지/업체 정보 입력칸 추가
- 입력한 홈페이지 정보에서 원장/대표 소개, 철학/상담 기준, 병원/업체 장점 후보 정리
- GPTs 초안 프롬프트에 홈페이지 정보 반영 규칙 삽입
- 마무리에서 홈페이지에 실제로 있는 정보만 1~2문장으로 반영하도록 제한
- 홈페이지 정보가 없으면 본원/저희/철학/장점 문구 임의 생성 금지 유지
