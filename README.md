# 달로썸 원고 검수기 v3.9

v3.8의 자료조사·도입설계 기능에 **감정 유지 / 문단별 고민 배치** 기능을 추가한 버전입니다.

## 핵심 추가 기능

- A등급 공통팩트는 본문 설명용으로 유지
- B등급 고민패턴을 도입부에만 쓰지 않고 본문 전환부에도 배치
- 문단별 고민 배치안 자동 생성
- GPTs용 초안 프롬프트에 “감정 유지 규칙” 자동 포함
- Claude용 지시문에 “감정/고민 문장 삭제 금지” 자동 포함

## 작업 흐름

1. GPT 조사 프롬프트 생성
2. GPT 조사 후 링크 직접 확인
3. 확인된 조사 결과를 원고 설계 모드에 붙여넣기
4. 자료등급 / 고민패턴 / 도입화법 / 문단별 고민 배치안 확인
5. GPTs용 초안 프롬프트 생성
6. GPTs 초안 작성
7. 원고 검수
8. Claude 말맛 다듬기
9. 최종 검수 후 제출

## 실행

```bash
pip install -r requirements.txt
streamlit run app.py
```
