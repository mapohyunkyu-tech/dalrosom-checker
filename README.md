# 달로썸 원고 검수기 v5.8

## 이번 버전 핵심

v5.8은 초보 작성자가 가장 헷갈리는 **키워드 횟수·위치·누락 문단 보완**을 기준으로 강화한 버전입니다.

### 추가 기능

1. **0. 기본 납품 설정 / 키워드 배치 기준**
   - 작업 모드 선택
   - 제목 키워드 필수 여부
   - 본문 키워드 최소/최대 횟수
   - 전체 키워드 최소/최대 횟수
   - 첫 문단 키워드 필수 여부
   - 마지막 문단 키워드 필수/권장/상관없음
   - 소제목 카운트 포함/제외
   - 정확히 일치 / 띄어쓰기 차이 인정 / 변형 허용
   - 한 문단 키워드 최대 횟수 설정

2. **키워드 배치 지도**
   - 제목, 도입, 본문, 마무리에 키워드를 어디에 넣을지 안내
   - GPT 조사 프롬프트와 GPTs 초안 프롬프트에 같이 반영

3. **원고 검수 모드 키워드 배치표**
   - 제목 키워드 횟수
   - 도입/본문1/본문2/본문3/마무리별 키워드 횟수
   - 소제목 횟수와 본문 횟수 분리
   - 본문 기준/전체 기준 카운트 표시

4. **누락 문단 키워드 삽입 요청문 생성**
   - 본문2 같은 특정 문단에 키워드가 빠졌을 때 전체 재작성하지 않도록 지시문 생성
   - “해당 문단 안에 키워드 1회만 자연스럽게 추가”하도록 제한

## 실행 방법

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 사용 흐름

1. ① GPT 조사 프롬프트에서 기본 납품 설정과 홈페이지 URL/업체명을 함께 설정합니다.
2. GPT 조사 결과를 확인하고 ② 원고 설계 모드에 붙여넣습니다.
3. ② 원고 설계 모드에서 화법, 도입 방식, 첫문장 형태, 키워드 배치 기준을 확정합니다.
4. GPTs용 초안 프롬프트를 생성합니다.
5. ③ 원고 검수 모드에서 키워드 배치표와 누락 문단 삽입 요청문을 확인합니다.
