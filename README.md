# 달로썸 원고 검수기 v5.3 B등급 고민 기반 감정문장 버전

v5.2의 화법 선택 / 도입 첫문장 형태 선택 기능을 유지하면서, 감정 도입·본문 브릿지·마무리 재연결 문장이 반드시 조사자료의 **B등급 잠재고객 고민패턴**에서 나오도록 강화했습니다.

## 핵심 수정

- GPT 조사 프롬프트 `[6-1] 선택 화법 실제 적용 설계` 강화
  - 감정 도입 첫문장 후보마다 `근거 B등급 고민` 표시
  - 본문 고민 브릿지 후보마다 `근거 B등급 고민` 표시
  - 마무리 재연결 문장 후보마다 `근거 B등급 고민` 표시
- GPTs용 초안 프롬프트에 `[B등급 고민 기반 감정문장 필수 규칙]` 추가
- 자료에 없는 증상, 감정, 비용 불안, 선택 상황을 GPT가 임의로 만들지 못하게 지시
- A등급은 팩트용, B등급은 감정/고민 문장용, C등급은 말맛 참고용으로 역할 분리 강화
- B등급 고민이 부족하면 구체 공감문을 억지로 만들지 말고 기준 확인형으로 처리하도록 수정
- 화면 버전명 v5.3으로 수정

## 실행 방법

```bash
pip install -r requirements.txt
streamlit run app.py
```
