# 달로썸 검수기 v7.5

## 변경 핵심

홈피드형을 전체 카테고리에 적용되는 별도 엔진으로 분리했습니다.
기존 검색형/업체형처럼 `주제 → 핵심 키워드 → 업체명 → 전환 목표`로 가지 않고, 아래 흐름으로 작동합니다.

1. 카테고리 선택
2. 최근 이슈·신상·가격·논란·반응 수집
3. 국내자료와 해외자료 분리 정리
4. 글감으로 쓸 이슈 1개 선택
5. 제목·썸네일·첫문장·본문으로 홈피드화

## v7.5 수정 내용

- 홈피드 카테고리에 `패션/명품/잡화`, `IT/가전/디지털`, `여행/맛집/핫플`, `소비/절약/가격비교` 흐름을 강화했습니다.
- 관심 입력칸을 `관심 브랜드/상품/인물/팀/키워드`로 변경했습니다.
- 샤넬 신상, 국내외 가격 비교, 다이소 품절템, 스포츠 이슈, 코인/경제 이슈처럼 카테고리별 글감 수집이 가능하도록 예시를 바꿨습니다.
- 해외 수집 범위에 `국내외 가격·출시 비교`를 추가했습니다.
- 해외 소스에 공식 브랜드/제조사/판매처, 글로벌 공식몰/국가별 가격 페이지, 패션 전문매체, IT 전문매체를 추가했습니다.
- ② 원고 설계 탭에서 홈피드형은 A/B/C 자료등급, 키워드 배치 지도, 업체 정보 구조를 쓰지 않고 `홈피드형 초안 프롬프트`를 바로 생성하도록 분리했습니다.
- 홈피드형 초안 프롬프트는 조사 결과에서 이슈 1개를 선택하고, 국내/해외 비교, 반응 포인트, 제목 후보, 썸네일 문구, 첫문장, 본문 초안, 이미지 방향까지 출력하도록 만들었습니다.

## 사용 방향

홈피드형은 주제와 핵심 키워드를 먼저 고정하지 않습니다.
예를 들어 `패션/명품/잡화` 카테고리에서 관심 단서에 `샤넬 신상, 국내외 가격`을 넣으면, 국내 출시 반응과 해외 공식 정보·가격 비교를 수집한 뒤 홈피드형 글감으로 바꾸는 구조입니다.

스포츠도 축구만이 아니라 야구, 농구, 배구, 격투기, 골프, e스포츠, 국제대회, 선수 인터뷰, 이적·트레이드, 부상, 팬 반응까지 카테고리 전체에서 수집합니다.

핵심은 `이슈 수집 → 글 내용 정리 → 홈피드화`입니다.
