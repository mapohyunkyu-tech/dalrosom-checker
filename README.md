# 달로썸 원고 검수기 v10.0.9

## 업데이트 내용

v10.0.9는 초안 생성 프롬프트의 분량 조건 우선순위를 보정한 핫픽스입니다.

### 수정 사항

- 초안 생성 프롬프트 상단에 `[분량 최우선 지시 · 반드시 준수]` 추가
- 조사 프롬프트에도 분량 최우선 지시 추가
- 기존 `[분량 조건]`을 `[분량 최우선 조건]`으로 강화
- `분량이 짧으면 본문을 압축한다` 문구를 `자연스럽게 보강한다`로 수정
- 목표 분량의 90%~110% 허용 범위와 85% 미만 금지 기준을 프롬프트에 명시
- 제목 후보, 최종 제목, 본문 라벨, 체크리스트 기호는 실제 본문 분량과 별도라는 문구 추가
- 2000자 조건에서 960자처럼 짧게 나오는 현상을 줄이도록 초안 작성 지시 0번에 분량 준수 지시 추가
- 작성 지시 번호 중복 일부 정리

## 테스트

- `python -m py_compile app.py` 통과
- `run_menu_fake_test.py` 통과
- `run_integration_harmony_check.py` 통과
- `run_100_sample_backtest.py` 통과

## 사용 메모

2000자 내외 원고를 만들 때는 ②에서 희망 분량을 2000자로 설정한 뒤 ③에서 생성된 GPTs 초안 프롬프트를 그대로 사용하세요. v10.0.9부터 분량 조건이 프롬프트 상단에 강하게 들어갑니다.
