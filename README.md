# 달로썸 원고 검수기 v6.2

## 변경 사항

- 원고 사용처 선택 모드 추가
  - 블로그 정보성
  - 카페 정보성
  - 커뮤니티/바이럴
  - 후기/리뷰
- 문단 구조는 공통으로 유지하고, 사용처별로 말투·문장 길이·홍보 허용도·마무리 방식을 조정
- 카페 정보성 모드
  - 회원이 정보 정리해서 공유하는 말투
  - 홍보성 문장, 가짜 후기 문장 금지
- 커뮤니티/바이럴 모드
  - 흥미 유도는 허용하되 낚시성·허위정보 금지
- 후기/리뷰 모드
  - 실제 경험 자료가 있을 때만 후기형 허용
  - 경험 자료가 없으면 후기인 척하지 않고 정보형으로 작성
- 병원/법률/금융 분야는 사용처와 별개로 위험표현 차단 규칙 유지
- v6.1의 자동추천 우선순위, v6.0의 검수조건 자동 불러오기 기능 유지

## 실행

```bash
pip install -r requirements.txt
streamlit run app.py
```
