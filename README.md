# 달로썸 원고 검수기 v4.3 제목유형 선택 버전

## 추가 기능

- 제목 6가지 방식 선택 추가
  1. 숫자/데이터 활용형
  2. 질문형
  3. 긴급성/한정성 강조형
  4. 궁금증 자극형
  5. 반전/의외성형
  6. 독자 상황 콕집기형

- 제목 기본 기준 내장
  - 핵심 키워드 제목 맨 앞 1회 우선
  - 제목 30자 이내 권장
  - 자극적 어그로가 아니라 독자의 호기심과 실제 고민을 건드리는 제목
  - 본문 내용과 다른 제목 금지

- GPT 조사 프롬프트에 제목 유형/제목 후보 요청 반영
- 원고 설계 모드에서 제목 유형 자동 추천 + 직접 선택 가능
- GPTs용 초안 프롬프트에 선택 제목 유형과 제목 후보 반영
- Claude용 지시문에 제목 유형과 키워드 앞 배치 유지 조건 반영
- 원고 검수 모드에서 제목 유형 감지/불일치 체크

## 실행

```bash
pip install -r requirements.txt
streamlit run app.py
```
