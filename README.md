# 달로썸 원고 검수기 v10.0.12

## 소제목 출력 강제 핫픽스

이번 버전은 GPTs 초안에서 본문 소제목이 사라지거나 문단에 붙어 나오는 문제를 줄이기 위한 패치입니다.

### 수정 내용
- 초안 프롬프트에 `[소제목 출력 필수 규칙]` 블록 추가
- 2000자 내외 원고는 본문 소제목 4~6개를 한 줄 단독으로 출력하도록 강제
- 도입 체크리스트는 소제목 수로 세지 않도록 안내
- 소제목이 본문 문장에 붙는 예시를 금지 예시로 추가
- ⑧ 최종 원고 검수에서 1700자 이상 원고는 소제목 최소 4개 미만이면 더 강하게 감점
- 소제목이 본문과 같은 줄에 붙어 있을 가능성 감지

### 검수
- `python -m py_compile app.py` 통과
