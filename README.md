# 달로썸 원고 검수기 v3.8

## 추가 기능

- GPT 조사 프롬프트 생성
- A등급 공통 핵심정보 추출 기준 포함
- B등급 잠재고객 고민패턴 추출 기준 포함
- C등급 말맛 참고 기준 포함
- 원고 설계 모드
- 도입 화법 자동 추천
- GPTs용 초안 프롬프트 생성
- Claude용 윤문 지시문 생성
- 기존 v3.7 검수 기능 유지

## 실행

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 사용 흐름

1. `① GPT 조사 프롬프트`에서 주제/키워드 입력 후 프롬프트 복사
2. GPT에 붙여넣고 조사 요청
3. GPT가 준 링크를 직접 확인
4. 확인된 자료 요약을 `② 원고 설계 모드`에 붙여넣기
5. 프로그램이 GPTs용 초안 프롬프트 생성
6. GPTs 초안 작성
7. `③ 원고 검수 모드`에서 1차 검수
8. Claude용 지시문으로 말맛만 다듬기
9. 최종 검수 후 90점 이상이면 잠금
