# 달로썸 원고 검수기 v10.0.16

v10.0.16 대표자 주장형 강화 패치입니다.

## 핵심 변경
- 대표자 주장형이 선택되거나 보조 글결로 들어갈 때 실제 본문에 전문가/대표 관점 문장이 보이도록 필수 장치를 추가했습니다.
- 오해 반박형, 장비보다 해석/판단 강조형, 선택 기준형 등 보조 글결도 검수에서 함께 확인하도록 수정했습니다.
- 일반 GPT용 프롬프트에 [글결 필수 장치]를 추가해 달로썸 GPTs 없이도 대표원장 칼럼형, 오해 반박형, 장비보다 판단 강조형이 더 잘 나오도록 강화했습니다.
- Claude 보강 패키지도 대표자 주장형/오해 반박형/장비보다 해석형 보강 지시를 자동 포함합니다.
- v10.0.15의 비표준 어미 감지와 전문직 문체 안전장치는 유지했습니다.
