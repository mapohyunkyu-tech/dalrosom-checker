# 달로썸 원고 검수기 v7.8

## 업데이트 요약

v7.8은 홈페이지 정보 입력 시 마무리 반영을 필수화한 버전입니다.

### 수정 사항

1. 홈페이지 정보 입력 시 마무리 반영 필수
   - 공식 홈페이지 또는 사용자가 입력한 홈페이지 정보가 있으면 확인된 장점·철학·상담 기준을 마무리에 반드시 1~2문장 반영하도록 변경했습니다.
   - 단, 입력 자료에 없는 장점, 철학, 경력, 장비, 수상 이력은 만들지 않습니다.

2. 공식 홈페이지 조사 프롬프트 강화
   - URL이나 업체 단서가 있을 때 공식 채널에서 확인된 정보만 분류합니다.
   - 주제와 연결 가능한 내용은 마무리 반영 후보로 반드시 정리합니다.
   - 관련 정보가 부족하면 “마무리 반영 가능 정보 부족”이라고 표시하도록 했습니다.

3. 병원/의료 대표 작성형 규칙 보강
   - 공식 홈페이지 정보가 제공되면 확인된 병원 장점·철학 중 주제와 연결되는 내용을 마무리에 반영합니다.
   - 치료 효과 보장, 최고/유일, 부작용 없음 같은 표현은 계속 금지합니다.

4. 초안 작성 프롬프트 보강
   - 마무리 작성 지시에 홈페이지 정보 필수 반영 조건을 추가했습니다.
   - 강한 홍보가 아니라 확인된 상담 기준이나 장점을 짧게 연결하도록 했습니다.
