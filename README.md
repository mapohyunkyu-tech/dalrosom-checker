# 달로썸 원고 검수기 v3.5 - 보수적 점수 보정

v3.4의 달로썸 8가지 도입 방식은 유지하되, 점수가 과하게 높게 나오는 문제를 보정한 버전입니다.

## v3.5 수정점

- 도입 방식 불일치 감점 강화
- 체크리스트형 선택 후 실제 체크리스트가 없으면 총점 상한 88점 적용
- 마무리 철학 반영 선택 후 철학 문장 부족 시 총점 상한 92점 적용
- 공백 제외 1,500자 미만이면 5만 원 이상 포트폴리오급 제한
- 비교 표 활용 오감지 수정: 실제 표 형태가 있어야 인정
- 알짜 정보 예고 오감지 수정: 검색/알짜/놓치기 쉬운 정보 예고가 명시돼야 인정
