# 달로썸 원고 검수기 v10.0.15

## 업데이트 요약

v10.0.15는 달로썸 GPTs에만 의존하지 않고, 일반 GPT에 프롬프트를 그대로 붙여도 글결·분량·전문직 문체가 무너지지 않게 보강한 버전입니다.

### 핵심 변경

1. **일반 GPT용 프롬프트 모드 추가**
   - ③ 원고 설계의 프롬프트 출력 방식 기본값을 `일반 GPT용`으로 변경했습니다.
   - 달로썸식 기존 말맛이 필요할 때만 `달로썸 GPTs용`을 선택하면 됩니다.
   - 외부 GPTs에는 `외부 GPTs용 강제 프롬프트`를 사용할 수 있습니다.

2. **글결 강제 실행 규칙 추가**
   - 일반 GPT용 프롬프트에 `[일반 GPT용 실행 규칙]`이 들어갑니다.
   - 대표 글결과 보조 글결이 제목, 도입, 소제목, 마무리에 보이도록 강제합니다.
   - 단순 체크리스트 정보글로 축소되는 문제를 줄였습니다.

3. **전문직 문체 안전장치 추가**
   - 병원·법률·전문직 원고에서 `입니다요`, `합니다요`, `됩니다요`, `필요합니다요` 같은 비표준 어미를 금지합니다.
   - 기본 종결을 `~입니다`, `~합니다`, `~할 수 있습니다`, `~확인해야 합니다` 중심으로 고정합니다.

4. **앱 검수에서 비표준 어미 감지**
   - 최종 검수에서 비표준 어미가 나오면 `AI티 > 비표준 어미`로 감지합니다.
   - 비표준 어미가 있으면 문체 오류 상한을 적용합니다.

5. **Claude 패키지에도 문체 안전장치 반영**
   - Claude가 윤문하면서 이상한 구어체 어미를 남기거나 새로 만들지 않도록 지시를 추가했습니다.

## 권장 사용법

- 기본 작업: `일반 GPT용` 프롬프트 생성 → 일반 GPT에 붙여 초안 작성
- 달로썸 기존 말맛 필요: `달로썸 GPTs용` 선택
- 남의 GPTs 사용: `외부 GPTs용 강제 프롬프트` 선택
