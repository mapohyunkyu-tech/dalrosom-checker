# 달로썸 원고 검수기 v4.5

## 외부 GPTs 강제 프롬프트 + 복사 찌꺼기 검사 버전

v4.4에 아래 기능을 추가한 버전입니다.

### 변경 사항

- 원고 설계 모드에 `프롬프트 출력 방식` 추가
  - 달로썸 GPTs용
  - 외부 GPTs용 강제 프롬프트
- 제목 조건 강화
  - 키워드 제목 앞 1회 우선
  - 30자 이내 권장
  - 제목 후보 3개 제시 후 최종 제목 1개 사용 지시
  - 선택한 제목 유형을 필수 조건으로 반영
- 도입 8가지 방식 강제 규칙 추가
  - 선택한 도입 방식은 추천이 아니라 도입부 필수 구조로 반영
  - 비교표 선택 시 도입부 초반에 마크다운 표 필수
  - `|---|---|---|` 구분선 필수 지시
- 출력 정리 규칙 추가
  - `+1`, `[1]`, `cite`, `source`, `reference`, `�` 등 검색/복사 찌꺼기 제거 지시
  - 홈페이지 정보 없을 때 “저희 병원은”, “본원은” 등 임의 철학 생성 금지
- 검수 모드에 복사찌꺼기 항목 추가
  - 깨진 문자
  - 출처 +1 흔적
  - 각주 번호 흔적
  - cite/source/reference/utm 흔적

### 사용 권장

- 네가 만든 GPTs 또는 달로썸 GPTs: `달로썸 GPTs용`
- 남의 GPTs, 일반 GPT, Claude 초안 작성용: `외부 GPTs용 강제 프롬프트`
