# 달로썸 원고 검수기 v6.5

## 변경 사항

- `작성자 관점`을 ① 의뢰 조건 입력 화면 앞으로 이동했습니다.
- ② 원고 설계 화면에서도 ① 입력값을 사용할 때 작성자 관점이 같이 자동 적용됩니다.
- ③ 원고 검수 모드에서 작성자 관점이 더 이상 기본 `에스테틱 원장`으로 고정되지 않고 ①/② 선택값을 우선 불러옵니다.
- 분야/사용처별 기본 작성자 관점을 추가했습니다.
  - 에스테틱 / 피부관리 → 에스테틱 원장
  - 병원 / 의료 → 피부과 원장/전문의
  - 법률 → 변호사
  - 학원/교육, 청소/홈케어 등 → 전문업종 대표
  - 카페/커뮤니티/후기형 → 정보성 블로그 작성자
- GPT 조사 프롬프트, GPTs 초안 프롬프트, Claude 윤문 지시문에 `작성자 관점`이 명시되도록 수정했습니다.
- ② 조사 결과 붙여넣기·원고 설계 화면의 별도 `홈페이지 정보 반영` 입력칸을 제거했습니다.
- 업체 공개정보·플레이스·홈페이지 내용은 ②의 `초안 작성 추가 조건` 또는 `GPT 조사 결과` 안에 넣어 반영하도록 정리했습니다.
- 홈페이지 정보가 없어도 조사 결과/추가 조건에 확인된 업체 공개정보가 있으면 그 범위 안에서만 반영할 수 있도록 안내문을 완화했습니다.

## 사용법

1. ①에서 주제, 키워드, 분야, 사용처, 작성자 관점을 먼저 정합니다.
2. GPT 조사 프롬프트를 만들어 조사합니다.
3. ②에는 조사 결과만 붙여넣고, 업체 공개정보가 필요하면 `초안 작성 추가 조건`이나 조사 결과 안에 넣습니다.
4. GPTs 초안 작성 후 ③ 원고 검수 모드에 원고를 붙여넣고 검수합니다.
5. 검수 결과 아래 Claude 지시문을 복붙해 윤문합니다.
