# 달로썸 원고 검수기 v6.6

## 변경사항

- ① 의뢰 조건 입력 화면에 다음 항목을 전면 배치했습니다.
  - 글 성격
  - 말투 세부 지시
  - 보조 키워드
  - 업체명/브랜드명
  - 전환 목표
  - 업체명 반영 강도
- ② 조사 결과 붙여넣기·원고 설계에서 ① 입력값을 그대로 불러오고, 브랜드·말투 조건을 확인할 수 있게 했습니다.
- GPT 조사 프롬프트, GPTs용 초안 프롬프트, Claude 윤문 지시문에 글 성격/브랜드/전환/말투 조건이 모두 반영됩니다.
- 일반 정보성 글과 대표 작성형 정보성 글이 섞이지 않도록 다음 구분을 추가했습니다.
  - 일반 정보성
  - 대표 작성형 정보성
  - 매출 전환형 정보성
  - 홈판 후킹형
- ② 화면의 별도 홈페이지 입력칸은 유지하지 않고, 필요한 업체 공개정보는 초안 작성 추가 조건이나 조사 결과에 넣는 방식으로 유지했습니다.

## 실행 방법

```bash
pip install -r requirements.txt
streamlit run app.py
```
