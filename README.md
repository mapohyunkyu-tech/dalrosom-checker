# 달로썸 원고 검수기 v5.9

## 이번 버전 핵심

v5.9는 ①에서 이미 입력한 의뢰 조건을 ② 원고 설계 모드에서 다시 쓰지 않도록 흐름을 정리한 버전입니다.

### 수정 기능

1. **② 조사 결과 붙여넣기·원고 설계**
   - `① 의뢰 조건 입력값 그대로 사용` 옵션 추가
   - 주제, 핵심 키워드, 분야, 분량, 문단 설정, 키워드 납품 설정을 ①번 값으로 자동 적용
   - ②번에서는 GPT 조사 결과만 붙여넣고 바로 원고 설계 가능

2. **분량 입력 혼란 정리**
   - 희망 분량이 `1000자/1500자/2000자/3000자`일 때는 직접 입력 글자수 칸을 숨김
   - `직접 입력`을 선택했을 때만 직접 입력 글자수 칸 표시
   - 조사/설계/검수 모드 모두 동일하게 정리

3. **탭 이름 정리**
   - ① 의뢰 조건 입력·GPT 조사 프롬프트
   - ② 조사 결과 붙여넣기·원고 설계
   - ③ 원고 검수 모드

4. **v5.8 기능 유지**
   - 기본 납품 설정
   - 키워드 배치 지도
   - 문단별 키워드 배치표
   - 누락 문단 키워드 삽입 요청문
   - 홈페이지 URL 조사 지시
   - 화법/도입 첫문장/B등급 고민 기반 검수

## 실행 방법

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 사용 흐름

1. ①에서 실제 의뢰 조건과 키워드 납품 기준을 입력합니다.
2. 생성된 조사 프롬프트를 GPT에 붙여넣습니다.
3. GPT 조사 결과를 ②에 붙여넣습니다.
4. ②에서 `① 의뢰 조건 입력값 그대로 사용`을 켠 상태로 GPTs용 초안 프롬프트를 생성합니다.
5. ③에서 완성 원고를 검수합니다.
