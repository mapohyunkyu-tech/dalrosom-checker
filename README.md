# 달로썸 원고 검수기 v7.0

## 업데이트 요약

v7.0은 네이버 홈피드형 원고 모드에 해외자료 수집/활용 옵션을 추가한 버전입니다.

### 주요 변경점

1. 홈피드형 해외자료 옵션 추가
- 해외자료 포함 여부: 포함 안 함 / 관련 있을 때만 포함 / 반드시 포함
- 해외자료 활용 방식: 원문 팩트 확인용 / 해외 반응 참고용 / 국내 기사와 비교용 / 제목 후킹 소재 참고용 / 종합 활용

2. 홈피드형 자료 수집 방향 강화
- 국내자료는 한국 독자 반응, 댓글 분위기, 홈피드 소재 확인용
- 해외자료는 최초 보도, 원문 팩트, 해외 반응, 깊은 맥락 확인용
- 스포츠, 주식/코인, 해외연예, IT 이슈는 관련 있을 때 해외 원문 확인 권장

3. 해외자료 안전장치 추가
- 한 해외 매체의 단독 보도나 추측은 확정 표현 금지
- 번역투 복붙 금지
- 국내 독자가 이해하기 쉬운 말로 재구성
- 원문 링크 자료 목록에 남기도록 지시

4. 기존 기능 유지
- 업체/검색형 원고
- 대표 작성형 정보성
- 매출 전환형 정보성
- 홈피드 후킹형
- 주식/코인/투자 안전장치

## 사용 예시

스포츠 홈피드 글에서는 국내 뉴스와 댓글 반응을 보고, 관련 해외 원문이나 외신이 있으면 함께 수집해 이슈 흐름과 팬 반응을 비교하는 방식으로 사용할 수 있습니다.
