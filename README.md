# 달로썸 원고 검수기 v4.4

## 제목 유형 선택 안함 추가 버전

v4.3의 제목 유형 선택 기능에 `선택 안함` 옵션을 추가한 버전입니다.

### 변경 사항

- 제목 유형 선택 항목에 `선택 안함` 추가
- `선택 안함` 선택 시 6가지 제목 유형을 강제하지 않음
- 그래도 제목 기본 기준은 유지
  - 핵심 키워드 제목 앞 1회 우선
  - 가능하면 30자 이내
  - 본문 내용과 다른 어그로 금지
  - 독자 고민과 호기심을 건드리는 제목 권장
- 검수 모드에서 `선택 안함`이면 제목 유형 불일치 감점 제외
- GPT 조사 프롬프트, GPTs 초안 프롬프트, Claude 윤문 지시문에 반영

### 사용 흐름

1. 제목 유형을 직접 고르고 싶으면 6가지 중 선택
2. 클라이언트가 제목 방식까지 정하지 않았거나 자연스럽게 가고 싶으면 `선택 안함`
3. 프로그램은 제목 기본 기준만 적용하고, 특정 유형은 강제하지 않음
