# 달로썸 검수기 v10.0.32

## 통합판정 국어선생님 2차검수 연동 핫픽스

이번 버전은 v10.0.30의 국어선생님 2차 재검수 로직을 ⑪ 통합 출고 판정에도 연결한 핫픽스입니다.

### 수정 내용

- ⑩ 국어선생님 최종 퇴고본을 ⑪ 통합 출고 판정에 그대로 넣으면 2차 재검수로 자동 인식합니다.
- ⑪ 통합판정이 국어선생님 결과를 다시 1차 원고처럼 채점해 85점권으로 떨어뜨리던 문제를 수정했습니다.
- 2차 재검수 상태에서는 같은 문체 지적을 반복하지 않고, 납품 사고급 표현만 다시 확인합니다.
- ⑪ 화면과 TXT 다운로드에 `국어선생님 2차검수: 적용/미적용` 표시를 추가했습니다.

### 사용법

1. ⑩ 국어선생님에서 원고를 퇴고합니다.
2. ⑩의 `최종 문단`을 ⑪ 통합 출고 판정에 붙여넣습니다.
3. ⑪ 실행 시 같은 원고로 판별되면 2차검수로 처리됩니다.
4. 납품 사고급 표현이 없으면 반복 감점 없이 통합판정이 진행됩니다.
