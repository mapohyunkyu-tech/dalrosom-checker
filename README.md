# 달로썸 원고 검수기 v4.1

## 변경점
- v4.0에서 원고 설계 모드에만 있던 분량 입력칸을 `① GPT 조사 프롬프트` 탭에도 추가했습니다.
- 이제 처음 조사 프롬프트를 만들 때부터 `희망 분량`, `공백 제외/포함`, `문단 설정`이 함께 들어갑니다.
- GPT 조사 결과 마지막에 분량/문단 반영 방향도 정리하도록 프롬프트가 강화되었습니다.

## 사용 순서
1. ① GPT 조사 프롬프트 탭에서 주제, 키워드, 분야, 희망 분량을 입력합니다.
2. 생성된 조사 프롬프트를 GPT에 붙여넣고 조사합니다.
3. 링크를 직접 확인한 자료만 ② 원고 설계 모드에 붙여넣습니다.
4. 설계 모드에서도 테스트 분량을 다시 확인하고 GPTs용 초안 프롬프트를 생성합니다.
5. 초안 작성 후 ③ 원고 검수 모드에서 점수와 위험표현을 확인합니다.
