# 달로썸 원고 검수기 v9.4

v9.4는 v9.3 쇼츠 대본 엔진에 `⑮ 홈피드 제목·썸네일 실험실`을 추가한 버전입니다.

## v9.4 추가 기능

- 홈피드 제목 후보 20개 자동 생성
- 썸네일 문구 후보 자동 생성
- 첫문장 후보 자동 생성
- 제목 각도 선택
  - 웃픈 현실형
  - 의외성형
  - 가격 반응형
  - 팬심 자극형
  - 논쟁형
  - 비교형
  - 저장형 정보형
  - 나만 그런가 공감형
- 홈피드 점수 산정
  - 이슈성
  - 공감성
  - 댓글 유도성
  - 저장 가치
  - 사진/썸네일 힘
  - 지금성
  - 논쟁 가능성
  - 정보 뒷받침
- 홈피드 프롬프트 자동 생성
- 홈피드 초안 간이 검수
  - 제목 과다 길이
  - 썸네일 문구 길이
  - 첫문장 피드 정지력
  - 보장/과장 제목
  - 본문 홈피드 포인트 약함
  - 검색형/AI식 문장
  - 이미지 저작권 위험
- 홈피드 발행/납품 포맷 자동 생성
- 홈피드 작업 로그 임시 저장

## 검증

- `python -m py_compile app.py` 통과
